# Место Россия
Адаптивный одностраничный сайт о красивых местах России.  
Проектная работа №5 в Яндекс.Практикуме на курсе «Веб-разработчик».

## Функциональность
- Адаптивная вёрстка (от 320px до 1280px и выше)
- Grid-Layout для карточек мест (1–3 столбца в зависимости от ширины экрана)
- Попап редактирования профиля с возможностью изменить имя и информацию о себе
- Переполнение текста в карточках и полях профиля с многоточием (text-overflow: ellipsis)

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, адаптивная и мобильная вёрстка)
- Nested БЭМ (структура файлов и нейминг классов)
- JavaScript (открытие/закрытие попапа, обработка формы, обновление данных на странице)
- Git
